🛒 Aplicación Web de Venta de Productos
📌 Descripción

Este proyecto consiste en una aplicación web de venta de productos desarrollada con React.
La aplicación simula una tienda online en la que los usuarios pueden ver productos, registrarse, iniciar sesión y gestionar un carrito de compra.

La web es responsive, por lo que se adapta a distintos tamaños de pantalla como ordenadores, tablets y móviles.

🎯 Objetivo del proyecto

Aplicar los conocimientos de React.

Trabajar con una base de datos y un backend.

Implementar un sistema de registro y login de usuarios.

Crear una aplicación web completa similar a una tienda online real.

🧩 Funcionalidades
Usuarios no registrados

Ver la página principal.

Ver el listado de productos.

Buscar productos por nombre o descripción.

Ver el detalle de un producto.

Al intentar añadir un producto al carrito se les pedirá iniciar sesión.

Usuarios registrados

Iniciar y cerrar sesión.

Añadir productos al carrito.

Modificar la cantidad de productos.

Eliminar productos del carrito (con confirmación).

Ver el precio total del carrito.

⚙️ Instalación y ejecución
Requisitos

Node.js

npm

Git

1️⃣ Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio

2️⃣ Instalar dependencias del frontend
cd frontend
npm install
npm start


La aplicación se ejecutará en:

http://localhost:3000

3️⃣ Backend y base de datos

El backend se ejecuta en local.

Se incluye un script de base de datos con datos de prueba o una configuración de Firebase.

Las credenciales se configuran en un archivo .env.
